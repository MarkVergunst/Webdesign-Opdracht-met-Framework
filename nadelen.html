Nadelen

De nadelen van Skelaton:

  • De CSS heb je zelf niet geschreven
  • Je weet niet precies wat je gaat verwachten voordat je stukken code hebt geschreven.
  • Je moet goed uitzoeken wat het precies doet omdat de website aardig beperkt is.
  • Heel weinig voorbeelden omdat het waarschijnlijk nog erg nieuw is.
  • Ze hebben geen standaard nav bar die ze laten zien op de website.
  • Website is niet direct responsive

dit zijn een aantal redenen waarom ik niet weer voor dit Framework zou kiezen.
