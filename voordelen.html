Voordelen

De voordelen van skeleton zijn dat het gemakkelijk gebruikt kan worden zonder dat je het moet installeren. De codes zijn makkelijk te begrijpen en geven veel ruimte om ze zelf aan te passen. Er zijn heel veel verschillende soorten styling die je kan gebruiken op je html bestanden met behulp van CSS. Zelf vind ik de Forum styling het mooiste omdat de layout van je pagina netjes voor je klaar staat en ook op de zelfde hoogste staat. Dit maakt het organiseren van websites een stuk makkelijker. Ook hun Media styling is erg handing sinds de website wordt aangepast op welk toestel je zit. Zoals dat je er kan voor zorgen dat de je telefoon precies op de juiste schermresolutie zet. Voor de rest vind ik dit een goede Framework voor kleine projecten die je het een stuk makkelijker maakt om je website te designen.
